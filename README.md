Exploratory Data Analysis of Netflix Movies and TV Shows catalog. Comprehensive analysis of content trends, distributions, and patterns.

##  Project Overview

This project performs in-depth Exploratory Data Analysis (EDA) on Netflix's content catalog to uncover insights about:
- Content distribution between Movies and TV Shows
- Temporal trends in content addition
- Genre popularity and patterns
- Production countries analysis
- Rating distributions

## Quick Start

### Prerequisites
- Python 3.8+
- pip package manager

### Installation
```bash
# Clone the repository
git clone https://github.com/regina31535-svg/netflix-eda-project.git
cd netflix-eda-project

# Install dependencies
pip install -r requirements.txt

# Launch Jupyter Notebook
jupyter notebook

# Open and run: notebooks/netflix_eda.ipynb

Dataset
Source: [Netflix Movies and TV Shows on Kaggle](https://www.kaggle.com/datasets/shivamb/netflix-shows)

Records: 8,807 movies and TV shows

Period: Data up to April 2021

Attributes: 12 attributes including type, title, director, cast, country, release year, rating, duration, and genres
